#Câu 6: Trình bày ý nghĩa toán tử /, //, %, **, and, or, is.

#1. /	Chia thực	
# Ví dụ: 10 / 4	= 2.5
#2. //	Chia lấy phần nguyên	10 // 4	2
#3. %	Chia lấy dư	
# Ví dụ : 10 % 4	= 2
#4. **	Lũy thừa	
# Ví dụ: 2 ** 3	8
#5. and	Logic AND 
# Ví dụ : True and False	False
#6. or	Logic OR	
#  False or True
# Ví dụ :	True
#7. is	So sánh định danh
# Ví dụ :	a is b	True/False
